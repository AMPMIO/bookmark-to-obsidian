## Summary

Introducing ExampleLib — zero-config JSON querying for the terminal

Tired of piping everything through jq? ExampleLib gives you a simple, readable syntax for filtering and transforming JSON from any source.

Features:
- Zero dependencies
- Pipe-friendly: `cat data.json | ex '.users[].name'`
- 10x faster than alternatives on large files
- Works with JSONL, ndjson, and streaming input

GitHub: https://github.com/example_dev/examplelib
